Mission 2 : Déploiement d'une Infrastructure Dédiée

Problématique : L'échec du test local

Pour gagner en autonomie vis-à-vis d'EVE-NG et passer à l'automatisation, j'ai choisi la technologie Containerlab. J'ai d'abord tenté de déployer cet environnement sur mon poste de travail (PC Portable Pro).

Le constat fut immédiat : Les images de routeurs (vJunos/cRPD), bien que conteneurisées, demandent beaucoup de ressources. Avec mes 16 Go de RAM, je ne pouvais lancer que 2 routeurs avant saturation. Impossible donc de simuler une topologie complexe de 5 à 10 routeurs.

La Solution : Un Serveur "Bare Metal" Dell R730

Pour lever cette contrainte matérielle, j'ai récupéré et remis en état un serveur d'entreprise. L'objectif : disposer de ma propre plateforme de calcul pour héberger mes labs Docker sans limite de ressources.

Preuve : Le serveur Dell R730 une fois racké et câblé par mes soins.

Actions Techniques Réalisées :

  • Installation Physique : Rackage dans la baie, câblage réseau uplink (10 Gbps) et alimentation redondée.
  • OS & Système : Installation d'Ubuntu Server 20.04 LTS. J'ai privilégié un OS Linux natif (pas de virtualisation type ESXi) pour maximiser les performances de Docker.
  • Sécurisation : Configuration d'un accès distant via Bastion SSH pour administrer le serveur depuis mon poste de travail en toute sécurité.

Preuve de fonctionnement

Une fois le serveur opérationnel, j'ai vérifié les ressources disponibles. Cette étape a marqué la fin des limitations matérielles et le début de la phase d'automatisation (Mission 3).

# Connexion au serveur via le Bastion
youenn@pc-pro:~$ ssh -J bastion.orange.intra youenn@192.168.10.5
youenn@dell-r730:~$ htop
# Résultat : 128 Go de RAM disponibles, 32 cœurs CPU.
# Le serveur est prêt à accueillir les topologies massives.

Bilan de l'étape

Cette mission m'a permis de valider des compétences Système & Hardware (AC22.02). J'ai compris qu'avant de coder (Software), il faut une infrastructure (Hardware) solide et dimensionnée. Je suis désormais autonome sur mon propre environnement.
